# Генератор статей на Python

Эта программа на Python использует нейронные сети ChatGPT 3.5 Turbo и Dall-e 2 для генерации статей и обложек к ним. Сгенерированные статьи сохраняются в файл и добавляются в папку States.

## Установка и запуск

1. Установите Python 3.
2. Клонируйте репозиторий на свой компьютер.
3. Установите необходимые библиотеки, введя команду:
```
pip install -r requirements.txt
```
4. Запустите программу, введя команду:
```
python generate_articles.py
```

## Использование

При запуске программы, она будет запрашивать параметры для генерации статьи и обложки. Пользователь может ввести название статьи, категорию, тему и длину текста в словах. Программа также попросит пользователя ввести ключевые слова, которые будут использоваться для генерации обложки.

По окончании генерации, сгенерированная статья сохраняется в файл с названием, которое задал пользователь. Обложка сохраняется в ту же папку, что и статья, с названием в формате "название статьи_cover".
